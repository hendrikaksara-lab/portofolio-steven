# Personal Blog

Blog pribadi untuk berbagi artikel tentang pemrograman dan teknologi.

## Fitur
- Daftar artikel
- Kategori (Pemrograman, Web Development, Tips)
- Pencarian artikel
- Sistem komentar (demo)
- Desain responsif

## Cara Menjalankan
Buka file `index.html` di browser web Anda.

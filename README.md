# Portfolio React

This is a React version of the portfolio website, built with Vite.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to `http://localhost:5173` to view the app.

## Build for Production

```bash
npm run build
```

## Troubleshooting

- If you encounter issues with dependencies, try deleting `node_modules` and running `npm install` again.
- Ensure Node.js version 16+ is installed.
